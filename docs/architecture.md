## Shared Components

```
TradingLogic (src/common/trading_logic.py)
в”ңв”Җв”Җ should_buy()          вҖ” threshold + confidence + position sizing
в”ңв”Җв”Җ should_sell_on_sentiment() вҖ” bearish threshold check
в”ңв”Җв”Җ check_stop_loss()     вҖ” fixed or trailing SL
в”ңв”Җв”Җ check_take_profit()   вҖ” fixed TP
в””в”Җв”Җ update_peak()         вҖ” trailing SL peak tracking

  вҖў Backtest engine (portfolio_backtest.py)
  вҖў Live trader (news_trader.py вҶ’ SentimentTrader)
```

- Maximum loss is capped at initial capital
